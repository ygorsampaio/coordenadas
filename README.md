# Coordenadas

Projeto de coordenadas com frontend e backend.

## Estrutura

- `frontend/`: Arquivos do frontend (HTML, CSS, JS)
- `backend/`: API backend com Node.js e MongoDB

## Como executar

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
Abra `[frontend/index.html](https://coordenadas-6uck.vercel.app/)` no navegador ou use um servidor local.
